# iStack Simple Website Project Template

## Requirements

- Download and Install Node - https://nodejs.org/download/
- Download and Install GitBash - https://git-scm.com/downloads

## First Time Install
- Intall global modules for BOWER and GULP
```bash
$ npm install gulp -g
$ npm install bower -g
```


## Quick Start
 
```bash
$ npm install
$ bower install
$ npm start
```
